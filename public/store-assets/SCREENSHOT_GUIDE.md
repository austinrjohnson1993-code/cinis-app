# Cinis App Store Screenshots Guide

## Required Screens to Capture

1. **Dashboard** — Tasks tab with tasks visible, "What's my next move?" button
2. **Check-in** — Active AI conversation showing persona coaching
3. **Habits** — Habit cards with streaks visible
4. **Journal** — Journal entry in progress with prompt selector
5. **Finance** — Budget tab with income breakdown
6. **Settings** — Persona selector showing 3 coaching styles
7. **getcinis.app** — Landing page hero

## Apple Required Sizes

- **6.9" iPhone** (1320 × 2868 px) — required
- **6.5" iPhone** (1242 × 2688 px) — required
- **5.5" iPhone** (1242 × 2208 px) — required
- **12.9" iPad Pro** (2048 × 2732 px) — if supporting iPad

## Google Play Required Sizes

- **Phone**: 16:9 or 9:16, min 320px, max 3840px
- **Minimum 2 screenshots, maximum 8**
- **Feature graphic**: 1024 × 500 px

## Tools

- Use Chrome DevTools device emulation for consistent sizing
- Set viewport to iPhone 14 Pro Max (430 × 932) for 6.9" captures
- Zoom browser to 100% before capturing
- Use browser screenshot tool or CleanShot X

## Copy Overlays

Each screenshot should have:
- Short headline at top (ember color, Playfair Display)
- Device frame (optional but premium)
- Coal background behind device
